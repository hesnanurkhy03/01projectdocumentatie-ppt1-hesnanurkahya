Week 5 – HTML & CSS experiment

Ontwerp testen in code en basiskennis verdiepen

In week 5 heb ik geëxperimenteerd met HTML en CSS om te testen hoe mijn treinscherm-ontwerp er zou uitzien wanneer het wordt omgezet naar code. Ik heb mijn ontwerp laten proberen uitwerken via ChatGPT, puur als test en om te begrijpen hoe een visueel ontwerp technisch kan worden opgebouwd.

Het resultaat kwam niet volledig overeen met mijn originele ontwerp. De verhoudingen, spacing en typografie klopten niet helemaal, waardoor het geheel er druk en minder overzichtelijk uitzag. Toch was dit experiment nuttig om te zien waar de verschillen zitten tussen grafisch ontwerp en technische uitwerking.

Naast deze test heb ik ook verschillende video’s opgezocht over HTML en CSS om de basis beter te begrijpen. Hierdoor kreeg ik meer inzicht in hoe layouts, blokken en stijlen worden opgebouwd in code.

Reflectie: Deze week heeft mij geholpen om te begrijpen dat een goed ontwerp niet automatisch goed vertaald wordt naar code. Het experiment motiveerde mij om HTML en CSS beter te leren en bewuster na te denken over technische haalbaarheid.
